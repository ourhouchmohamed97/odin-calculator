# Calculator

A browser-based calculator built with HTML, CSS, and JavaScript as part of **The Odin Project Foundations** curriculum.

## About

This project recreates a standard calculator that performs the four basic arithmetic operations. It features a clean user interface, supports chained calculations, handles edge cases, and demonstrates JavaScript event handling and DOM manipulation.

## Features

- Addition, subtraction, multiplication, and division
- Interactive button-based interface
- Real-time display updates
- Decimal number support
- Chained calculations (e.g. `12 + 7 - 1 =`)
- Clear button to reset the calculator
- Prevents consecutive operator evaluations
- Starts a new calculation after displaying a result
- Rounds long decimal results
- Handles division by zero with an error message

## Built With

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (ES6)
